# 🚀 Déploiement sur Vercel\n\n## Guide d'installation et déploiement sur Vercel\n\n### ✅ Prérequis\n\n1. **Compte Vercel** : https://vercel.com (gratuit)\n2. **GitHub connecté** : Votre repo doit être sur GitHub\n3. **CLI Vercel** (optionnel) :\n   ```bash\n   npm install -g vercel\n   ```\n\n---\n\n## 📝 Étapes de déploiement\n\n### Option 1 : Déploiement via GitHub (Recommandé)\n\n#### 1. **Connecter Vercel à GitHub**\n   - Allez sur https://vercel.com/new\n   - Cliquez sur \"Import Git Repository\"\n   - Autorisez Vercel à accéder à votre compte GitHub\n   - Sélectionnez le repository `movie_recommendations`\n\n#### 2. **Configuration du projet**\n   - **Framework** : \"Other\" (car c'est Flask, pas Node.js)\n   - **Build Command** : `pip install -r requirements.txt`\n   - **Output Directory** : Laissez vide\n   - **Install Command** : `pip install -r requirements.txt`\n\n#### 3. **Variables d'environnement**\n   - Vous pouvez en ajouter si nécessaire (ici, aucune n'est obligatoire)\n\n#### 4. **Cliquez sur \"Deploy\"**\n   - Vercel va cloner votre repo\n   - Installer les dépendances\n   - Déployer l'app\n   - Vous recevrez une URL en direct ! 🎉\n\n---\n\n### Option 2 : Déploiement via CLI Vercel\n\n```bash\n# 1. Naviguez dans le dossier du projet\ncd movie_recommendations\n\n# 2. Connectez-vous à Vercel\nvercel login\n\n# 3. Déployez\nvercel --prod\n\n# 4. Suivez les instructions\n```\n\n---\n\n## 📊 Structure pour Vercel\n\n```\nmovie_recommendations/\n├── api/\n│   └── index.py                 ← Entry point Flask\n├── templates/\n│   └── index.html               ← Frontend\n├── static/\n│   ├── css/style.css\n│   └── js/script.js\n├── data/                        ← À uploader après deploy\n│   ├── ratings.csv\n│   └── movies.csv\n├── requirements.txt\n├── vercel.json                  ← Configuration Vercel\n├── .vercelignore\n└── app.py                       ← Local seulement\n```\n\n---\n\n## 🔧 Configuration Vercel\n\nLe fichier `vercel.json` configure :\n- **Builds** : Comment construire l'app (Python + static)\n- **Routes** : Routage des requêtes vers `/api/index.py`\n- **Environment** : Variables d'env (FLASK_ENV=production)\n\n```json\n{\n  \"version\": 2,\n  \"builds\": [\n    { \"src\": \"api/index.py\", \"use\": \"@vercel/python\" }\n  ],\n  \"routes\": [\n    { \"src\": \"/(.*)\", \"dest\": \"/api/index.py\" }\n  ]\n}\n```\n\n---\n\n## 📂 Ajouter les données après déploiement\n\n### Via Vercel Storage (Recommandé)\n1. Allez dans votre projet Vercel\n2. Onglet **Storage**\n3. Créez une **Blob Storage**\n4. Uploadez `ratings.csv` et `movies.csv`\n\n### Ou via Git (Simple)\n1. Ajoutez les fichiers CSV au repo (modifiez `.gitignore`)\n2. Commitez et poussez\n3. Vercel re-déploiera automatiquement\n\n```bash\n# Modifiez .gitignore\n# Commentez ou supprimez la ligne \"data/\"\n\ngit add data/ratings.csv data/movies.csv\ngit commit -m \"Add MovieLens data\"\ngit push origin testing\n```\n\n---\n\n## ✨ Après déploiement\n\n### 1. **Vérifiez le statut**\n   - Votre app sera accessible via : `https://your-project-name.vercel.app`\n   - Testez l'endpoint `/api/status` pour vérifier les données\n\n### 2. **Domaine personnalisé (optionnel)**\n   - Onglet **Domains** dans Vercel\n   - Connectez votre domaine DNS\n\n### 3. **Logs et débogage**\n   - Allez dans **Deployments** dans Vercel\n   - Cliquez sur votre déploiement → **Logs**\n   - Consultez les erreurs en temps réel\n\n---\n\n## 🐛 Troubleshooting\n\n### \"Module not found\"\n```bash\n# Les dépendances manquent\n# Vérifiez que requirements.txt est à la racine\n# et toutes les dépendances y sont listées\n```\n\n### \"CSV file not found\"\n```bash\n# Les données ne sont pas déployées\n# Uploadez ratings.csv et movies.csv dans le repo\n# ou utilisez Vercel Storage\n```\n\n### \"Cold start delay\"\n- C'est normal sur Vercel (serverless)\n- La première requête peut prendre 10-30 secondes\n- Après, c'est rapide ⚡\n\n---\n\n## 📱 Tester l'API en production\n\n```bash\n# Remplacez YOUR_URL par votre URL Vercel\ncurl https://YOUR_URL.vercel.app/api/status\n\n# Recommandations\ncurl -X POST https://YOUR_URL.vercel.app/api/recommendations \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\"user_id\": 1}'\n\n# Films populaires\ncurl https://YOUR_URL.vercel.app/api/top-movies\n```\n\n---\n\n## 🎯 URL de prod\n\nAprès déploiement, votre app sera disponible à :\n\n**https://movie-recommendations-jacques.vercel.app** (exemple)\n\n---\n\n## 📚 Ressources\n\n- [Vercel Docs -
